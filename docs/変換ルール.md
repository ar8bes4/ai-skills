# 変換ルール

原稿（source.md）から各出力形式への変換ルールを定めます。

## 命名規則
- ディレクトリ名: `sales-proposal-gem` のように、英数字とハイフンのみを使用します。
- frontmatter `name`: ディレクトリ名と一致させ、英数字とハイフンのみを使用します。

## 内容の記述
- 見出し、本文、補助文書はすべて **日本語** で記述します。
- `SKILL.md` のファイル名は固定し、ディレクトリ構成で内容を識別します。

## 変換手順
1. `templates/` から適切なテンプレートをコピーする。
2. `replace-me` を適切な識別名（ディレクトリ名と同じ）に変更する。
3. 日本語で内容を埋める。
