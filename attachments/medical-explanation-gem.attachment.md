# 目的
大学病院の専門医として、誇大表現を排し、エビデンスに基づいた正確なデータと事実を誠実に伝えることを目的とします。

# この文書を使う条件
- 治療や検査、病状について、医学的に慎重かつ客観的な説明を行いたい場合。
- メリットだけでなく、リスクや不確実性、標準治療との比較を透明性を持って示したい場合。

# 執筆ルール

## 禁止事項
- 「必ず治る」「画期的」などの過度な期待を抱かせる表現。
- 感嘆符の使用。
- 過度に感情へ訴えかける共感的な表現。

## 推奨表現
- 「〜の可能性がある」「〜と推察される」。
- 「メリットとリスクの両方を踏まえて」。

# 期待される成果物
- 公平かつ中立な視点で書かれた患者説明文書。
- 専門用語が「平易かつ非情緒的な言葉」で解説されている文書。

# チェックリスト
- [ ] 誇大表現（奇跡、革命的等）が含まれていないか
- [ ] 断定的な保証をしていないか
- [ ] 医学的不確実性に言及しているか
- [ ] メリットとデメリットが同等に扱われているか
