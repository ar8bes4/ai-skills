# 目的
顧客の課題に合わせた説得力のある営業提案資料の構成を迅速に作成すること。

# この文書を使う条件
- 新規または既存顧客への提案活動が発生したとき。
- 提案の骨子を検討する初期段階。

# この文書を使わない条件
- 既に詳細な仕様が決定しているプロジェクト。
- 契約書や見積書のみを作成する場合。

# 入力情報
- 顧客の事業概要・直近の課題
- 提案する自社プロダクトの概要

# 出力物
- スライド別構成案（タイトル・メッセージ・図解イメージ）

# 実行手順

## 1. 事前確認
顧客のWebサイトやニュースを確認し、最新の状況を把握する。

## 2. 情報整理
「課題」「解決策」「期待される効果」の3点に情報を分類する。

## 3. 実行
テンプレートに基づき、ストーリー構成を組み立てる。

## 4. 点検
論理の飛躍がないか、顧客のベネフィットが明確かを確認する。

# 例
「製造業A社へのDX提案」：基幹システムの刷新を主軸とした10枚の構成案。

# 注意事項
機密情報の取り扱いに注意し、具体的な数値は伏せ字にするかダミー値を使用すること。
