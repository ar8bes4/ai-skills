# fact-check-gem

世界最高峰の調査報道ジャーナリスト兼シニア・ファクトチェッカーとして、情報の正確性を厳格に検証し、詳細な監査レポートを作成するスキルです。

## 主要ファイル
- **[SKILL.md](file:///c:/Users/yert1/Documents/git/ai-skills/.agents/skills/fact-check-gem/SKILL.md)**: スキルの実効定義。
- **[examples/sample-case.md](file:///c:/Users/yert1/Documents/git/ai-skills/.agents/skills/fact-check-gem/examples/sample-case.md)**: 医学的な臨床解析レポートの検証実例。

## 特徴
- 100点満点でのエグゼクティブサマリー形式のスコアリング
- 詳細な検証テーブル（真、疑い、誤りの判定）
- 専門的な根拠（引用・データ）の提示
