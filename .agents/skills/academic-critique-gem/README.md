# academic-critique-gem

学術雑誌のエディターの視点で、文章を徹底的に批判し、論理的整合性の取れた完璧な文章へリライトするためのスキルです。

## 主要ファイル
- **[SKILL.md](file:///c:/Users/yert1/Documents/git/ai-skills/.agents/skills/academic-critique-gem/SKILL.md)**: スキルの実効定義。
- **[examples/sample-case.md](file:///c:/Users/yert1/Documents/git/ai-skills/.agents/skills/academic-critique-gem/examples/sample-case.md)**: 学術的な文章添削の具体例。

## 役割
- 提供された文章の論理的矛盾を指摘
- 根拠の薄弱な主張を特定
- 洗練された学術的トーンへの変換
