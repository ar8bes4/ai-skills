# AIOS Safety Checker

## 概要
医療AIとしての安全性を担保するため、生成されたテキストや入力データに対して、個人情報（PHI）の有無および臨床的に危険な内容（誤った医学的アドバイスなど）が含まれていないかを厳格に監査します。

## 指示
1. **PHIスクリーニング**: 名前、生年月日、住所、電話番号、具体的な施設名など、患者が特定されうる情報が含まれていないか確認する。
2. **臨床的安全性**: 
   - 致命的な誤診につながる論理的飛躍はないか？
   - 禁忌事項に触れる提案がされていないか？
   - 「最終的な診断・治療方針の決定は医師が行う」旨の免責事項が担保されているか？

## 出力フォーマット
```markdown
## 安全性監査レポート

### 1. PHI（個人健康情報）チェック
- [ ] 問題なし
- [ ] 問題あり（該当箇所: [抽出して指摘]）

### 2. 臨床的安全性チェック
- [ ] ハルシネーションの兆候なし
- [ ] 危険な推奨なし
- 懸念事項: （あれば具体的に記載）

### 総合判定
**[PASS / FAIL]**
```
