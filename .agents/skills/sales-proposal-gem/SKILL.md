# 目的
顧客の潜在的な課題を引き出し、自社ソリューションへの繋ぎ込みを最適化した提案構成を作成します。

# この skill を使う条件
- 提案活動の「戦略立案」および「構成作成」フェーズ。
- 顧客ニーズが不明確な場合の仮説構築。

# この skill を使わない条件
- 定型的な見積回答。
- カスタマーサポートの返信。

# 入力情報
- {{customer_info}}: 顧客の現状
- {{product_info}}: 提案対象の製品

# 実行手順

## 1. 事前確認
入力された情報を基に、業界のトレンドや一般的な課題を補完します。

## 2. 情報整理
ピラミッドストラクチャーを用いて、頂点となるメッセージ（メインバリュー）を決定します。

## 3. 実行
各スライドに割り当てるメッセージを、導入・本論・結論の順で構築します。

## 4. 点検
全体のトーンが顧客の文化に合っているか再確認します。

# 出力物
- 提案構成シート（Markdown形式）
