# medical-explanation-gem

大学病院の専門医として、患者に対しエビデンスに基づいた客観的かつ誠実な情報提供を行うためのスキルです。

## 主要ファイル
- **[SKILL.md](file:///c:/Users/yert1/Documents/git/ai-skills/.agents/skills/medical-explanation-gem/SKILL.md)**: スキルの実効定義。
- **[examples/sample-case.md](file:///c:/Users/yert1/Documents/git/ai-skills/.agents/skills/medical-explanation-gem/examples/sample-case.md)**: 喉頭癌（高分化型）に対する誠実な説明文書の実例。

## 方針
- **宣伝文句の排除**: 「画期的」「奇跡の」などの誇大表現を禁止
- **透明性の確保**: メリット、リスク、不確実性を同等に扱う
- **平易な解説**: 専門用語を「非情緒的な平易な言葉」で補足
