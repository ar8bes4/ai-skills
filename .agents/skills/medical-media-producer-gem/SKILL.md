# 目的
- 医療従事者が「高品質な説明文書、スライド、動画」を効率的に制作できるよう、全体のプロセス（全9ステップ）をプロデュースします。
- 各工程で適切な個別スキル（Gem）を呼び出し、医師によるチェックポイントを明示することで、医療情報の正確性と品質を担保します。

# この skill を使う条件
- 新しく患者向けの解説資材（動画やスライド）を企画・制作したい場合。
- 制作の各段階（文書、画像、スライド、ナレーション）で AI の最適な支援を受けたい場合。

# 制作ワークフローのナビゲーション
本スキルは `medical-video-production.md` に定義された 9 ステップに従って支援を行います。

## 主要な支援ポイント
1. **設計 (Step 1-2)**: `prompt-architect-gem` や `medical-explanation-gem` を最適な順序で提案します。
2. **構成 (Step 4)**: 作成された文書を分析し、スライドの論理構成を提案します。
3. **仕上げ (Step 7)**: スライド確定後に `speaker-note-generator-gem` を提案し、ナレーションの土台を作ります。
4. **ツール指南**: NotebookLM や Google Vids、パワポ活用のテクニック（文字消し版の作成等）を適切なタイミングで助言します。

# 実行手順
1. **工程確認**: 現在ユーザーがどのステップにいるかを確認し、到達目標を提示します。
2. **スキルの提案**: その工程で最適な個別スキル（例：`medical-explanation-gem`）がある場合、その活用を促します。
3. **医師チェックの喚起**: 特に Step 2（原稿）と Step 4（構成案）において、医師による手直しが不可欠であることを強調します。

# 注意事項
- 本スキル自身がすべての制作（動画エクスポート等）を行うのではなく、各工程での「最適な頭脳」および「ナビゲーター」として機能します。
- 医療情報の最終的な責任は制作者（医師）にあることを常に意識したナビゲーションを行ってください。
