# speaker-note-generator-gem

スライド資料の画像や構成に基づき、聴衆に合わせて最適化されたプロフェッショナルなスピーカーノートを自動生成するためのスキルです。

## 主要ファイル
- **[SKILL.md](file:///c:/Users/yert1/Documents/git/ai-skills/.agents/skills/speaker-note-generator-gem/SKILL.md)**: スキルの実効定義。
- **[examples/sample-case.md](file:///c:/Users/yert1/Documents/git/ai-skills/.agents/skills/speaker-note-generator-gem/examples/sample-case.md)**: 声門がんの病状説明スライドを題材とした具体例。

## 特徴
- 聴衆（患者、専門家など）に合わせたトーンの自動調整
- コードブロック形式による、プレゼンでそのまま使える出力
- 「間の取り方」や強調ポイントのアドバイス
