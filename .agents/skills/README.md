## 構成
各スキルディレクトリ内には以下のファイルが含まれています：
- `SKILL.md`: スキルの核心となる実行定義。
- `README.md`: そのスキルの詳細な役割とファイル構成の解説。
- `examples/`: 成功事例（Few-shot リファレンス）。
