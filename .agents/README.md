# AI Skills & Workflows (.agents/)

このディレクトリは、AI アシスタント（Antigravity）が使用する高度なスキル（Gem）と標準ワークフローを管理するための中心的な場所です。

## ディレクトリ構造

### 1. skills/
特定の役割やタスクに特化した AI スキルの実行定義（SKILL.md）と実例（examples/）を格納しています。
- 各ディレクトリの `SKILL.md` が AI の行動指針となります。
- `examples/` には、AI の精度を向上させるための Few-shot 学習用具体例が含まれています。

### 2. workflows/
複雑な工程（動画制作、資料作成など）を段階的に進めるための標準手順書（.md）を格納しています。
- `/[workflow-name]` のようにスラッシュコマンドで呼び出すことが可能です。

## プロジェクトの管理原則
- **三層構造の維持**: 定義（skills/）、ソース（sources/）、添付用ドキュメント（attachments/）の整合性を常に保ちます。
- **Git による永続化**: すべての変更は Git を通じてチーム全体で共有されます。
