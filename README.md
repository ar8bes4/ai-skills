# AI Skills Repository

Gem や業務の流れを記載したファイルを `SKILL.md` に変換し、管理するためのリポジトリです。

## フォルダ構成

- `docs/`: 運用方針や変換ルールなどのドキュメント
- `sources/`: 元となる業務手順の原稿（source.md）
- `attachments/`: 添付ファイル用版のドキュメント
- `.agents/skills/`: Native Skill 版の `SKILL.md`
- `templates/`: 各パターンの雛形

## クイックスタート

1. `sources/` に新しいディレクトリを作成し、`source.md` を作成する
2. `templates/` の内容を参考に、`attachments/` と `.agents/skills/` にファイルを生成する
3. Git でコミットして管理する
